Illegal Variable Names:

1name, user-name, var, @value, first name

Rules for naming JS variables

Variable names can only contain, numbers, $ and For example: Smy_IstVariable

Variable must begin with a letter, 5 or For example: Sname, name or name

Variable names are case sensitive

Variable names should not be JS keywords
